# Linear-Regression

Three ML implementations using just numpy and matplotlib

## Gradient Descent Linear Regression
Learns a non-linear relationship between input and output variables using feature engineering and gradient descent.

- Implements batch gradient descent from scratch
- Engineers polynomial features to capture non-linearity
- Visualizes the learned model against training and test data

## Normal Equation
Solves the same regression problem as above without iteration or learning rate.

- Implements the closed-form solution: θ = (XᵀX)⁻¹Xᵀy
- Compares results with gradient descent approach

##  Hogsmeade Housing Prices
Predicts house prices for Hagrid using real estate data from Hogsmeade.

- Compares simple linear regression and locally weighted linear regression
- Analyzes which features have the most impact on predicted prices
- Reports average least squares error for each model
